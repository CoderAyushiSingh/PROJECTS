# Student Performance Prediction

## Problem Statement
Predict student performance based on academic and personal factors to enable early intervention.

## Dataset
Student performance dataset containing academic and demographic features.

## Approach
- Data cleaning and preprocessing
- One-hot encoding for categorical variables
- Train-test split
- Feature scaling for linear models
- Model training and evaluation

## Models Used
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

## Evaluation Metrics
- MAE
- RMSE
- R² Score

## Result
Random Forest performed best with lowest RMSE and highest R² score.

## Tools & Technologies
Python, Pandas, NumPy, Scikit-learn, Google Colab
